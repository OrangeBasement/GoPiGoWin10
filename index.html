GoPiGo for Windows 10 IoT

This project was created to allow .Net programmers to develop their own projects with the GoPiGo. The GoPiGo is a robotic car for the RaspberryPi created by www.dexterindustries.com. Below are several starter solutions to help you start building your next GoPiGo Windows 10 project.

GoPiGo Library

Base library that all the GoPiGo Win 10 rojects use. Enables commutation between with GoPiGos motor controller, servos, LEDs, and more.

GoPiGo-Controller and GoPiGo-Server

Client (Controller) and Server for the GoPiGo. Starter solution that allows control of your GoPiGo over a LAN connection or even the internet. The Controller provides a XAML touch screen interface to control movement, servos, and LEDs.

GoPiGo-HDMIUntethered: GoPiGo Client and Server on Windows 10 IoT

Similar to the GoPiGo Client/Server project, the GoPiGo-HDMI solution allows control of the GoPiGo from a XAML interface that is displayed on an HDMI device. The GoPiGo must have an HDMI device plugged in as well as a USB mouse for this to work. Project is more of a proof of concept of UI control with the GoPiGo

GoPiGo-Headless

GoPiGo-Headless is the starter project for an autonomous robot. Programmed to move randomly, this can be expanded on to react to sensors and other devices for a truly amazing robot that is self controlled.
